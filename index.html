SWITCH

SWITCH digunakan untuk memilih salah satu dari beberapa blok kode berdasarkan nilai ekspresi tertentu.


$day = "Monday";
switch ($day) {
    case "Monday":
        echo "It's Monday, start of the work week!";
        break;
    case "Friday":
        echo "It's Friday, time to relax!";
        break;
    default:
        echo "Have a nice day!";
}

Output: It's Monday, start of the work week!

Penjelasan: Nilai variabel $day diperiksa menggunakan switch. Berdasarkan nilai $day, kode dalam blok case yang sesuai akan dijalankan. Jika tidak ada nilai yang cocok, blok default akan dijalankan.

LOOPING

Looping adalah proses yang digunakan untuk mengulangi eksekusi blok kode tertentu selama suatu kondisi terpenuhi. Berikut adalah contoh dengan menggunakan for loop:


for ($i = 1; $i <= 10; $i++) {
    echo "Juara $i, ";
}

Output: Juara 1, Juara 2, Juara 3, ..., Juara 10,

Penjelasan: Kode ini menggunakan perulangan for untuk mencetak string "Juara" diikuti dengan angka dari 1 sampai 10.

WHILE

WHILE adalah jenis loop yang terus berjalan selama kondisinya bernilai true.


$count = 1;
while ($count <= 5) {
    echo "This is iteration number $count
";
    $count++;
}

Output:

This is iteration number 1
This is iteration number 2
This is iteration number 3
This is iteration number 4
This is iteration number 5

Penjelasan: Loop dimulai dengan mengevaluasi kondisi $count <= 5. Selama kondisi bernilai true, kode dalam loop akan dijalankan dan nilai $count akan bertambah setelah setiap iterasi.

DO WHILE

DO WHILE mirip dengan while, tetapi memastikan blok kode dijalankan setidaknya sekali sebelum memeriksa kondisinya.


$count = 1;
do {
    echo "This is iteration number $count
";
    $count++;
} while ($count <= 5);

Output:

This is iteration number 1
This is iteration number 2
This is iteration number 3
This is iteration number 4
This is iteration number 5

Penjelasan: Kode ini memastikan bahwa blok kode dijalankan setidaknya sekali, meskipun kondisi awal tidak terpenuhi.

FOR

FOR digunakan untuk mengulangi blok kode dengan jumlah iterasi yang ditentukan.


for ($i = 1; $i <= 5; $i++) {
    echo "This is iteration number $i
";
}

Output:

This is iteration number 1
This is iteration number 2
This is iteration number 3
This is iteration number 4
This is iteration number 5

Penjelasan: Loop for digunakan untuk menjalankan kode dengan jumlah iterasi yang sudah ditentukan, dari 1 hingga 5.

Kombinasi Contoh

Berikut adalah contoh implementasi beberapa fungsi sekaligus:


$day = "Monday";
switch ($day) {
    case "Monday":
        echo "Start of the work week!
";
        break;
    default:
        echo "Enjoy your day!
";
}
$count = 1;
while ($count <= 3) {
    echo "Iteration $count
";
    $count++;
}

Output:

Start of the work week!
Iteration 1
Iteration 2
Iteration 3

Penjelasan: Program pertama mencetak pesan berdasarkan nilai $day menggunakan SWITCH. Kemudian, melakukan perulangan dengan WHILE untuk mencetak iterasi dari 1 hingga 3.
